title: Menghapus file temporary ~
---

Ini membosankan harus selalu menghapus berkas <i>temporary</i> sebelum melakukan <code>git commit</code>, jika tidak maka berkas <i>temporary</i> yang berakhiran <code>~</code> akan ikut terunggah.

Secara standar, Linux selalu membuat berkas <i>temporary</i> ketika sebuah berkas di gubah, maksudnya untuk berjaga - jaga jika terjadi kesalahan dalam penggubahan, sehingga pengguna masih bisa kembali mempergunakan berkas aslinya. Sangat cerdas dan membantu.

Namun kadang kala hal ini mengganggu juga, apalagi jika lama tidak dibersihkan maka tentu akan memenuhi Hard Disk atau kadang ikut terunggah. Untungnya ada <i>script</i> yang bisa dipergunakan untuk mencari berkas - berkas tersebut, kemudian menghapusnya. Lewat Terminal ketik kode sebagai berikut :

<pre>
 <code>
  $ find . -name ‘*~’ -exec rm -r ‘{}’ \;
  </code>
</pre>

Dan viola, semua berkas <i>temporary</i> akan dicari dan dihapus. fnd
