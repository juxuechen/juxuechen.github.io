Di <a href="http://gnome.org">Gnome 3</a> atau Gnome-Shell, untuk membuka daftar aplikasi bisa dilakukan dengan cara meletakkan kursor <i>mouse</i> di atas panel yang bertuliskan <b>activities</b>. Dengan ini secara otomatis halaman aplikasi dan jendela aplikasi yang terbuka akan ditampilkan.

Di banyak tema, panel Activities hanya ditampilkan ala kadarnya, tampilan teks dengan efek gelombang jika kursor mouse lewat. Untungnya Gnome-Shell dibangun diatas HTML dan CSS, bagi web desainer ( atau yang paham dengan CSS ) tentu familiar dengan hal ini. Sehingga bisa dengan mudah melakukan modifikasi agar tampilan lebih indah.

Kali ini yang ingin dirubah adalah tampilan Panel Activities agar mempergunakan vertical-banner khusus, kebetulan saya bikin yang ada logo <a href="http://archlinux.org">Arch</a>nya. Caranya mudah, bisa dilakukan seperti ini :

<h2>Persiapkan banner pengganti</h2>

Bisa bikin sendiri atau cari di google, kemudian bisa dimodifikasi. Ukuran yang bagus adalah dengan lebar <code>58px</code>.

<h2>Edit file </code>gnome-shell.css</code></h2>

Letak file ini ada dalam satu folder dengan file tema yang dipergunakan, bisa di cari di <code>~/.themes</code>. Kemudian cari baris yang bertuliskan

{% highlight css %}
 #panelActivities {
   border: none;
   background-position: 0 0;
   width: 58px;
   height: 24px;
   color: rgba(0,0,0,0.0);
   position: absolute;
                    }
{% endhighlight %}
  
Kemudian tambahkan kode <code>background-image: url("banner.png");</code> di dalam kode CSS ID panelActivities. Sehingga akan menjadi kira - kira seperti ini :

{% highlight css %}
  #panelActivities {
    border: none;
    background-position: 0 0;
    width: 58px;
    height: 24px;
    color: rgba(0,0,0,0.0);
    position: absolute;
    background-image: url("banner.png");
                    }
{% endhighlight %}
  
<p>&nbsp;</p>  

<h2>Restart Gnome-Shell</h2>
  
Simpan file, dan restart Gnome-Shell dengan menekan tombol <code>ALT</code>+<code>F2</code>, kemudian ketik <code>r</code>. Dan dengan segera tampilan panel activities akan berubah. fnd
